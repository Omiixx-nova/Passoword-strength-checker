Password Strength Checker

A simple web application built with HTML, CSS, and JavaScript that checks the strength of a password in real time and provides visual feedback.

Features

Real-time password strength evaluation

Strength levels: Weak, Medium, Strong

Color indicators (red, yellow/orange, green)

Validation for empty input

Checklist for:

Minimum length

Letters

Numbers

Symbols

How It Works

Weak: Less than 6 characters

Medium: 6–10 characters with letters and numbers

Strong: More than 10 characters with letters, numbers, and symbols

Project Structure

index.html – UI structure

style.css – Styling and layout

script.js – Logic for password validation and strength calculation

How to Run

Download or clone the repository

Open index.html in a web browser

Start typing a password to see its strength

JavaScript (Vanilla)
